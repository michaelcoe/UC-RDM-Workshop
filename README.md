# Research Data Management Training

This lesson is a carpentries adaption of Kristin Briney's "[The Research Data Management Workbook](https://doi.org/10.7907/z6czh-7zx60)". This lesson will give an overview of ways to implement good research data management practice. It is purposely written to be general so that it can be adapted to each institution's needs. It is written in a way to be done within an hour.
